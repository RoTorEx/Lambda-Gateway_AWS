# Intro to AWS DynamoDB


Tutorial how to start work with [AWS DynamoDB](https://aws.amazon.com/getting-started/hands-on/create-manage-nonrelational-database-dynamodb/).

## List of control scripts using boto3 to work with DynamoDB

*create_table.py* - create table in DynamoDB.

*insert_items.py* - insert some data to DB.

*get_item.py* - get item from DB.

*query_items.py* - get query of items from DB.

*add_secondary_index.py* - create secondary global index to iem in DB.

*query_with_index.py* - collect items using secondary index.

*update_item.py* - update item in DB.

*delete_table.py* - delete table from DB.
